# 💰 포인트그라운드 (Point Ground)

혁명적인 130% 리워드 시스템 - 5명 그룹 완성 시 구매 금액의 130%를 보장하는 쇼핑 플랫폼

## ✨ 핵심 시스템

### 💯 130% 리워드 시스템

-   **5명 그룹 매칭**: 구매확정 순서대로 자동 매칭
-   **목표풀 설정**: 그룹 5명의 몸값(구매금액÷1.1×1.3) 합계
-   **재원 유입**: 6번째 사람부터 PV값이 지분 비율대로 분배
-   **자동 지급**: 누적풀이 목표풀 도달 시 130% 자동 지급

### 🎯 8개 라운드 시스템

1. **ROUND 1**: 10만원 ~ 20만원
2. **ROUND 2**: 20만원 ~ 30만원
3. **ROUND 3**: 30만원 ~ 50만원
4. **ROUND 4**: 50만원 ~ 100만원
5. **ROUND 5**: 100만원 ~ 300만원
6. **ROUND 6**: 300만원 ~ 1,000만원
7. **ROUND 7**: 1,000만원 ~ 3,000만원
8. **ROUND 8**: 3,000만원 이상

### 📱 주요 섹션

1. **히어로 슬라이더**: 130% 리워드 강조 (3개 슬라이드)
2. **핵심 시스템**: 구매 → 그룹 매칭 → 리워드 받기
3. **가격대별 라운드**: 8개 라운드 선택
4. **5명 그룹 시스템**: 작동 원리 상세 설명
5. **실시간 통계**: 완료 그룹, 리워드 지급액
6. **Before & After**: 실제 수익 인증
7. **비교 표**: 타 플랫폼 대비 우위
8. **실시간 리워드 지급 현황**: 라이브 피드
9. **사용자 후기**: 6개 실제 회원 리뷰

## 🛠️ 사용 방법

### 1. 파일 구조

```
쇼핑하면서 돈벌기/
├── index.html      # 메인 HTML 파일
├── styles.css      # CSS 스타일시트
├── script.js       # JavaScript 파일
└── README.md       # 이 파일
```

### 2. 실행 방법

1. **직접 열기**: `index.html` 파일을 브라우저에서 더블클릭
2. **Live Server** (VS Code):
    - Live Server 확장 설치
    - index.html에서 우클릭 → "Open with Live Server"
3. **Python 서버**:
    ```bash
    python -m http.server 8000
    ```
    브라우저에서 `http://localhost:8000` 접속

### 3. 커스터마이징
